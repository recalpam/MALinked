Malinked API

Version: 0.1 DEV (docs: 0.2) (May 11th 2015)

Index

  • GET /study - Get list of all studies
  • GET /study/{id} - Get single study
  • GET /study/{id}/groups - Get all groups from single study
  • GET /group - Get list of all groups
  • GET /group/{id} - Get single group
  • GET /group/{id}/students - Get all students from single group
  • GET /student/{id} - Get single student information
  • GET /search/{term} - Search for students by term
  • POST /auth - Generate authentication key for user login

Changelog

  • Added AUTH to docs and edited results for GET /study/{id}/groups and GET /group/{id}/students

Note: All the requests are refixed with /api/db in the url. So /search/{term} would be /api/db/search/{term}.
The part enclosed with brackets "{}" are GET parameters.
